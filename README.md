# NOTQ-Q1

منصة تعليمية مبنية باستخدام **React + Vite**  
تركّز على تدريب النطق والقراءة باللغة العربية، مع لوحات مخصصة للطالب، المعلم، وولي الأمر.

## التقنيات المستخدمة
- React
- Vite
- Tailwind CSS
- Supabase (Database & Auth)
- Vercel (Deployment)

## التشغيل محليًا

```bash
npm install
npm run dev

## Building the app

```bash
npm run build
```
